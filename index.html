HTML, CSS, JS & Tailwind Cara Membuat Website Kesan & Pesan

Mengenal HTML, CSS, JavaScript, dan Tailwind CSS

Dalam pengembangan website modern, keempat teknologi ini adalah pondasi utama yang membentuk tampilan dan fungsi dari setiap halaman web. Masing-masing punya peran penting yang saling melengkapi untuk menciptakan pengalaman pengguna yang interaktif, responsif, dan menarik.

HTML (HyperText Markup Language)

HTML adalah bahasa markup yang digunakan untuk menyusun struktur dasar halaman web. Elemen-elemen seperti teks, gambar, tautan, tabel, dan form didefinisikan menggunakan tag HTML. Bisa dibilang, HTML adalah ‚Äúkerangka tulang‚Äù dari sebuah website.

  • <h1> - <h6> : untuk judul dan subjudul.
  • <p> : untuk paragraf teks.
  • <a> : untuk membuat hyperlink.
  • <img> : untuk menampilkan gambar.
  • <div> : untuk mengelompokkan elemen.

CSS (Cascading Style Sheets)

CSS digunakan untuk mengatur tampilan visual dari halaman web. Dengan CSS, kita bisa mengubah warna, ukuran font, posisi elemen, animasi, dan banyak lagi. CSS menjadikan website terlihat menarik dan profesional.

  • color: mengatur warna teks.
  • background: memberi warna atau gambar latar belakang.
  • margin / padding: mengatur jarak antar elemen.
  • border-radius: membuat sudut elemen jadi melengkung.
  • animation: menambahkan efek bergerak.

JavaScript (JS)

JavaScript adalah bahasa pemrograman yang menambahkan interaktivitas pada website. Tanpa JS, web hanya akan bersifat statis. Dengan JS, kita bisa membuat tombol yang merespons klik, animasi dinamis, validasi form, kalkulator, sampai game berbasis web.

  • document.querySelector(): untuk memilih elemen HTML.
  • addEventListener(): mendeteksi aksi pengguna.
  • if/else, loop, function: logika pemrograman dasar.
  • fetch(): mengambil data dari API eksternal.

Tailwind CSS

Tailwind CSS adalah framework CSS modern yang menggunakan sistem ‚Äúutility-first‚Äù. Artinya, kita tidak perlu membuat file CSS terpisah ‚Äî cukup pakai class bawaan seperti bg-blue-500 atau text-center langsung di elemen HTML. Ini mempercepat proses styling dan menjaga konsistensi desain di seluruh halaman.

  • flex, grid: untuk layout responsif.
  • shadow-lg, rounded-xl: untuk efek bayangan dan sudut halus.
  • hover:text-blue-600: efek interaktif saat pointer diarahkan.
  • dark mode: dukungan tema gelap otomatis.

Tata Cara Membuat Website Sederhana

Membangun website sederhana itu sebenarnya gampang, asal ngerti urutannya. Berikut langkah-langkah dasar yang bisa untuk diikuti buat bikin web:

Buat Folder Proyek

Siapkan satu folder utama (misalnya myWebsite) sebagai tempat nyimpen semua file webmu. Folder ini kayak ‚Äúrumah besar‚Äù buat file HTML, CSS, gambar, dan JavaScript.

Buat Tiga File Utama

Bikin tiga file utama: index.html, style.css, dan main.js. Ketiganya punya fungsi berbeda ‚Äî HTML buat struktur halaman, CSS buat tampilan, dan JS buat interaksi.

Hubungkan File CSS & JS

Di dalam file index.html, tambahkan tag <link> buat nyambungin style.css dan tag <script> buat ngelink ke main.js. Supaya semua elemen bisa bekerja bareng. 

<!-- index.html -->
<!DOCTYPE html>
<html lang="id">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" 
          content="width=device-width, initial-scale=1.0" />
    <title>Website Sederhana</title>

    <!-- Hubungkan file CSS -->
    <link rel="stylesheet" 
          href="style.css" />
  </head>

  <body>
    <h1>Halo Dunia!</h1>

    <!-- Hubungkan file JavaScript -->
    <script src="main.js"></script>
  </body>
</html>

Percantik dengan CSS

Ubah warna, font, dan tata letak biar lebih estetik. Tambahin efek hover atau bayangan biar tampilannya lebih profesional.
